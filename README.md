# Cypress example

Cypress example

## Tech


* node.js - evented I/O for the backend
* Cypress - Fast, easy and reliable testing for anything that runs in a browser.


## Installation

Cypress example requires [Node.js](https://nodejs.org/) v10.15+ to run.

```sh
$ npm run setup
```
